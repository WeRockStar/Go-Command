```shell
	- go env - View environment
	- go install - Build and install 
	- go run <go_file>.go 
	- godoc <package_name> <function_name>
	- go version - View go version
	- go help - View help all command and option in go
	- go fmt - Format your code 
	- go imports - Insertion and removal of import declarations 
	- go doc
```
